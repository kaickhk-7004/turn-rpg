# ⚔️ 어둠의 탑 (turn-rpg)

모바일 브라우저에서 플레이하는 **정통 판타지 턴제 RPG** 웹 게임.

## 플레이 방법

GitHub Pages 배포 후 폰 브라우저에서 접속:

```
https://kaickhk-7004.github.io/turn-rpg/
```

> 배포 설정: 저장소 **Settings → Pages → Source: Deploy from a branch → `main` / `/ (root)`** 선택

로컬 실행:

```
python -m http.server 8000
# http://localhost:8000 접속
```

## 게임 소개

| 요소 | 내용 |
|---|---|
| 파티 | 전사 발더 🛡️ · 도적 실피 🗡️ · 마법사 메릴 🔮 · 성직자 루나 ✨ (4인 고정) |
| 진행 | 탑을 오르는 층(floor) 방식 — 층마다 3전투, **5층마다 보스** |
| 전투 | 속도 기반 턴제. 공격 / 스킬 / 물약 / 방어 |
| 성장 | 레벨업 시 능력치 상승 + **스킬 포인트 1** 획득 |
| 스킬트리 | 클래스별 2갈래 × 3티어. 이전 스킬을 배워야 다음 해금 |
| 장비 | 무기·방어구·장신구 3슬롯, 5등급: 일반 → 마법 → 레어 → 에픽 → **레전드** |
| 저장 | localStorage 자동 저장 (전투 후·장비 변경 시) |

### 장비 등급

| 등급 | 색 | 능력치 배율 | 부가 옵션 |
|---|---|---|---|
| 일반 | 회색 | ×1.0 | 0개 |
| 마법 | 파랑 | ×1.25 | 1개 |
| 레어 | 노랑 | ×1.55 | 2개 |
| 에픽 | 보라 | ×2.0 | 3개 |
| 레전드 | 주황 | ×2.6 | 4개 |

## 구조

```
index.html      화면 골격
css/style.css   모바일 우선 다크 판타지 테마
js/data.js      클래스·스킬트리·장비·몬스터 데이터
js/game.js      상태·성장·장비·저장
js/battle.js    턴제 전투 엔진 (버프/중독/적 AI)
js/ui.js        렌더링·입력 처리
```

빌드 도구 없는 순수 HTML/CSS/JS — 저장소를 그대로 정적 호스팅하면 동작한다.
